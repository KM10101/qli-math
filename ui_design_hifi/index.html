Qli Math 高保真原型

路线图驱动的个人数学学习知识库。这是高保真版本—— 真实内容、可交互的向量示意图、状态切换与笔记自动保存，采用 modern-minimal 视觉方向。 线框版本仍保留在 上一级目录。

核心链路（PRD §18.1）
首页→ 路线列表→ 路线详情→ 知识点详情

这四个页面决定产品主体验。知识点详情是核心，含可拖动的线性组合交互图。

P001

首页 Dashboard

当前路线、当前知识点、下一步建议、需复习与最近学习。
P002

学习路线列表

所有可学路线，含难度、阶段数、知识点数与进度。
P003

学习路线详情

目标 / 适合人群 / 前置，阶段式时间线与知识点状态。
P004

知识点详情（三栏）

左：路线目录 · 中：正文 + 交互图 · 右：状态/笔记/AI。
P105

我的学习

各路线进度、笔记、疑问、需复习与已掌握统计。
P106

知识库浏览

按领域分组浏览所有知识点的查阅入口。
P207

数学史

人物、事件、问题与思想演化的轻量探索入口。
账户08

登录

邮箱 / 密码登录，含校验、密码显示与登录链接入口。
账户09

个人设置

重设密码、通知邮箱、AI 模型与学习偏好，左侧分区切换。

关于这一版：所有页面共用 hifi/css/app.css 一套设计系统（颜色、排版、组件、状态符号 ○ ◐ ✓ !），便于统一维护。知识点详情页的状态切换与笔记会保存在浏览器本地。

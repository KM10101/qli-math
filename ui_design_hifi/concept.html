线性组合

线性组合就是用一组向量按比例相加，生成一个新的向量。

为什么要学

它是理解张成空间、线性相关、基和维数的起点。几乎所有线性代数的核心概念，都建立在「把向量按比例加起来」这一个动作之上——先把这个动作看清楚，后面的内容才不会变成符号堆砌。

直觉解释

给定两个向量 v1 和 v2，我们可以把它们各自缩放（乘以一个数），再相加。改变这两个缩放系数 a、b，得到的向量 av1 + bv2 就会落在平面上不同的位置。下面拖动滑块试试：

系数 a  (缩放 v₁)1.0
系数 b  (缩放 v₂)1.0
a·v1 + b·v2  =  (1.0, 3.0)

这里 v₁ = (2, 1)，v₂ = (-1, 2)。所有可能的落点，恰好铺满整个平面——这就是下一节「张成空间」的伏笔。

形式定义

给定向量 v1, v2, …, vn 与标量（实数）a1, a2, …, an，它们的线性组合定义为：

a1v1 + a2v2 + ⋯ + anvn

一个具体例子

取 v1 = 21，v2 = −12，计算 2·v₁ + (−1)·v₂：

缩放 v₁（×2）
2 · (2, 1) = (4, 2)
缩放 v₂（×−1）
−1 · (−1, 2) = (1, −2)
相加
(4, 2) + (1, −2) = (5, 0)

把上面交互图的滑块拨到 a=2, b=−1，结果向量正好落在 x 轴上的 (5, 0)。

常见误区

!
系数不必是正数。 它可以是任意实数，包括 0 和负数。系数为 0 表示「这个向量不参与」，为负表示「反方向」。
!
别把「线性组合」和「线性相关」混为一谈。 线性组合是一个动作（按比例相加）；线性相关描述的是一组向量之间的关系。两者紧密相连，但不是同一件事。

历史背景

向量与线性结构的现代表述，源自 19 世纪 Hermann Grassmann 的《延伸量理论》(1844) 与 Giuseppe Peano 在 1888 年给出的向量空间公理化。把向量「按比例相加」这一朴素动作抽象成线性运算，正是这一时期数学家从几何直觉走向代数结构的关键一步。

应用场景

机器学习

特征加权求和、词向量与嵌入的组合，本质都是线性组合——权重就是这里的系数。

计算机图形

颜色混合、坐标插值、动画补间，都是把若干向量按比例叠加得到的中间结果。
